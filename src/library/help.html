    怎么安装ipfs？

    第一步，下载：$wget https://dist.ipfs.io/go-ipfs/v0.4.13/go-ipfs_v0.4.13_linux-amd64.tar.gz
    第二步，解压：$tar -zxvf go-ipfs_v0.4.13_linux-amd64.tar.gz；移动：$cd go-ipfs；$sudo mv ipfs /usr/local/bin/ipfs
    第三步，新建节点：$ipfs init。
    第四步，启动节点：$ ipfs deamon

    项目怎么运行？

    第一步，启动ipfs: ipfs daemon
    第二步，启动ganache:./ganache ;windows就直接双击ganahce.exe即可。
    第三步，编译部署：truffle compile；truffle migrate。
    第四步，安装项目依赖(第一次运行项目)&运行：npm install;npm run dev
